
# Next.js 13 App

This is a Next.js 13 app. It is a great starting point for building your own web applications.





## getting started
```bash
git clone https://github.com/nayanprasad/next13-template

cd next13-template

yarn

yarn run dev

```
## Features

- TypeScript
- Tailwind CSS
- Eslint
- prettier
- Husky
